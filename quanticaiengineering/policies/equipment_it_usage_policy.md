## Policy Overview

### Purpose
This policy establishes guidelines for the use, care, and security of company-provided IT equipment and systems. Proper use of these resources ensures business productivity, protects company data, and maintains equipment lifecycle.

### Scope
This policy applies to:
- All employees, contractors, and temporary workers
- All company-owned equipment and IT resources
- Personal devices accessing company systems (BYOD)
- All locations (office, remote, travel)

### Covered Equipment and Systems
- Laptop and desktop computers
- Mobile devices (smartphones, tablets)
- Monitors, keyboards, mice, and peripherals
- Printers and scanners
- Networking equipment
- Software and applications
- Cloud services and SaaS platforms
- Company email and communication systems

### Employee Responsibilities
Employees are responsible for:
- Proper care and use of equipment
- Following security policies
- Reporting issues and damage promptly
- Returning equipment upon request or separation
- Using equipment primarily for business purposes

## Acceptable Use

### Business Use
Company equipment provided primarily for business purposes:
- Performing job responsibilities
- Professional development and training
- Business communication
- Authorized company business
- Reasonable personal use (see below)

### Personal Use
Limited personal use acceptable when:
- Outside work hours (or during breaks/lunch)
- Doesn't interfere with work responsibilities
- Not excessive in duration or frequency
- Complies with all policies (no illegal, offensive, or inappropriate content)
- Doesn't consume significant bandwidth or resources

**Examples of Acceptable Personal Use:**
- Checking personal email during lunch
- Brief personal phone call
- Online banking during break
- Shopping during off-hours

**Examples of Unacceptable Personal Use:**
- Streaming videos during work hours
- Personal business or side jobs
- Excessive social media browsing
- Gaming during work time
- Downloading large personal files

### Prohibited Activities
The following are strictly prohibited:
- Illegal activities
- Harassment, discrimination, or hostile conduct
- Viewing, storing, or transmitting inappropriate or offensive content
- Introducing malware or viruses
- Hacking or unauthorized access
- Circumventing security controls
- Peer-to-peer file sharing of copyrighted material
- Cryptocurrency mining
- Personal commercial activity or profit
- Excessive bandwidth consumption (large downloads, streaming)
- Anything violating other company policies

### Monitoring and Privacy
**Company Rights:**
- Monitor, access, and review equipment usage
- Track websites visited and applications used
- Review files stored on company devices
- Analyze network traffic and bandwidth usage
- Retrieve data from devices
- Disclose information when required by law

**Employee Privacy:**
- Employees have minimal expectation of privacy on company equipment
- Personal use should be limited and non-sensitive
- Highly personal or confidential matters should use personal devices
- Company respects employee privacy within legal and business constraints

## Software and Applications

### Approved Software
**Pre-Installed:**
- Operating system (Windows 11 or macOS)
- Microsoft Office 365 suite
- Web browsers (Edge, Chrome)
- VPN client (Cisco AnyConnect)
- Anti-malware software (Microsoft Defender)
- Communication tools (Teams, Outlook)

**Available Upon Request:**
- Development tools (Visual Studio, IntelliJ, etc.)
- Design software (Adobe Creative Cloud)
- Project management tools (Jira, Asana)
- Data analysis tools (Tableau, Power BI)
- Industry-specific applications

**Request Process:**
1. Check software catalog in IT portal
2. Submit software request ticket
3. Provide business justification
4. Manager approval may be required
5. IT reviews for security and licensing
6. Software installed or license provided

### Software Licensing
- Use only properly licensed software
- Don't install pirated or unauthorized software
- Don't share license keys
- One license per user (don't share accounts)
- Notify IT when software no longer needed (reclaim license)

### Personal Software
Installing personal software:
- Generally discouraged
- Must not violate any company policy
- Must not pose security risk
- Must not interfere with business software
- IT may remove personal software if necessary
- IT not responsible for supporting personal software

### Prohibited Software
Do NOT install:
- Unlicensed or pirated software
- Hacking or penetration testing tools (unless job requires and approved)
- Peer-to-peer file sharing applications
- Personal cloud storage (Dropbox personal, Google Drive personal)
- Cryptocurrency wallets or miners
- Remote access tools (unless approved by IT)
- Any software that violates security policy

### Cloud Services and SaaS
**Approved Services:**
- Office 365 (SharePoint, OneDrive, Teams)
- Salesforce
- Slack (if authorized by department)
- GitHub Enterprise
- AWS (via company account)
- Other IT-approved services

**Requesting New Services:**
- Submit request to IT with business case
- IT and Security review for:
  - Security and privacy compliance
  - Data sovereignty and location
  - Terms of service
  - Licensing and cost
  - Integration with existing systems
- Approval from IT Director required

**Prohibited:**
- Using unapproved cloud services for company data
- Signing up for services with company credit card without approval
- Storing confidential data in personal cloud accounts

### Software Updates
- Keep software up to date
- Install critical security updates immediately
- OS and application updates pushed automatically where possible
- Restart computer when prompted for updates
- Don't delay security updates

## Data Storage and Backup

### Approved Storage Locations
**Primary Storage:**
- OneDrive for Business (personal work files)
- SharePoint (team files and collaboration)
- Network drives (department shared folders)
- Approved cloud services (with IT approval)

**Backup:**
- OneDrive/SharePoint automatically backed up
- Network drives backed up nightly
- Local hard drive data may not be backed up
- Store critical data in OneDrive/SharePoint, not local drive

### Data Classification
Follow Data Privacy Policy classifications:
- **Public:** Marketing materials, public website content
- **Internal:** General business documents
- **Confidential:** Sensitive business information
- **Restricted:** Highly sensitive, regulated data

Store data in appropriate locations with proper security controls.

### Local Storage
**Laptop/Desktop Hard Drive:**
- Use for temporary working files
- Not a reliable long-term storage location
- Data may be lost if device fails
- Store important files in OneDrive/SharePoint

**External Drives:**
- Generally not recommended for business data
- If needed, request IT-approved encrypted drive
- Backup external drive data to OneDrive/SharePoint
- Keep external drives secure

### Prohibited Storage
Do NOT store company data on:
- Personal cloud storage (Dropbox personal, Google Drive personal, iCloud)
- Personal external hard drives
- USB flash drives (unless encrypted and approved)
- Personal computers
- Third-party file sharing sites

### Data Loss Prevention
- Save work frequently
- Use auto-save features
- Don't store important files only on local drive
- Test backups periodically
- Report data loss immediately to IT

### File Naming and Organization
- Use descriptive, searchable file names
- Include dates in format YYYY-MM-DD
- Avoid special characters (/ \ : * ? " < > |)
- Organize with logical folder structure
- Delete obsolete files regularly

## Internet and Network Usage

### Internet Access
- Provided for business purposes
- Reasonable personal use acceptable (see Acceptable Use)
- Usage monitored for security and policy compliance
- Bandwidth-intensive activities may be restricted

### Web Browsing
**Acceptable:**
- Work-related research and information
- Professional development
- Industry news and updates
- Reasonable personal browsing during breaks

**Prohibited:**
- Illegal content
- Adult or sexually explicit content
- Gambling sites
- Hate speech or extremist content
- Piracy or copyright infringement sites
- Sites known to distribute malware
- Excessive streaming video/music during work hours

### Website Blocking
- IT blocks access to prohibited categories
- Some sites blocked to prevent malware
- If legitimate business site blocked, request unblock via IT ticket
- Circumventing blocks prohibited

### Bandwidth Management
- Prioritize business-critical traffic
- Large downloads (>1GB) should occur during off-hours when possible
- Streaming video should be work-related and reasonable
- Report slow network issues to IT

### VPN Usage
**Required:**
- All remote access to company network
- When using public or untrusted Wi-Fi
- When accessing confidential information remotely

**VPN Client:**
- Installed on all company devices
- Connect to VPN before accessing company resources
- Keep VPN client updated
- Report VPN connection issues to IT

### Wi-Fi
**Office Wi-Fi:**
- Corporate network for company devices
- Guest network for visitors and personal devices
- Credentials provided by IT

**Home Wi-Fi:**
- Use secure, password-protected network (WPA3 or WPA2)
- Don't use open/unsecured networks
- Change default router password
- Keep router firmware updated

**Public Wi-Fi:**
- Avoid when possible
- Always use VPN if accessing company resources
- Disable file sharing and automatic connections
- Use mobile hotspot as alternative

## Equipment Care and Maintenance

### Physical Care
**General Guidelines:**
- Handle equipment carefully
- Keep away from liquids
- Don't eat or drink near keyboard
- Transport laptop in protective case
- Clean regularly with appropriate materials
- Avoid extreme temperatures

**Laptops:**
- Don't close lid with objects on keyboard
- Use surge protector
- Don't block ventilation openings
- Avoid placing on soft surfaces that block airflow
- Keep away from magnets

**Mobile Devices:**
- Use protective case and screen protector
- Keep charged (don't let battery fully deplete regularly)
- Avoid exposing to water (unless waterproof rating)
- Don't leave in hot car

### Cleaning
- Power off before cleaning
- Use microfiber cloth for screens
- Compressed air for keyboards
- Isopropyl alcohol wipes for surfaces (not on screens)
- Don't spray liquids directly on devices

### Maintenance and Updates
**Automatic:**
- OS and security updates
- Antivirus definition updates
- Some application updates

**User Responsibility:**
- Restart when prompted
- Install updates in timely manner
- Don't disable automatic updates
- Keep applications updated

**IT Maintenance:**
- Periodic security scans
- Performance optimization
- Warranty and repair management

### Troubleshooting
**Before Contacting IT:**
- Restart computer/device
- Check connections and power
- Check for updates
- Try different network/location
- Review error messages

**Contact IT For:**
- Persistent technical issues
- Hardware damage or malfunction
- Software installation or configuration
- Security concerns
- Data recovery

### Reporting Problems
**Non-Urgent Issues:**
- Submit ticket via IT portal: support.quanticai.com
- Email: ithelpdesk@quanticai.com
- Include detailed description and screenshots

**Urgent Issues:**
- Call IT Help Desk: Extension 5600
- Security issues: Extension 5200
- Business-critical outages receive priority

### Repairs and Replacements
**Minor Repairs:**
- IT assesses and repairs if feasible
- Warranty repairs processed by IT
- Loaner equipment available during repair

**Major Damage:**
- Assessment by IT
- Replacement if repair not cost-effective
- Employee may be responsible for cost if due to negligence or misuse

**Lost or Stolen Equipment:**
- Report immediately to IT and Security
- Police report required for stolen equipment
- Remote wipe performed to protect data
- Replacement provided
- Employee may be responsible for cost if due to negligence

## Return and Separation

### Return of Equipment
Equipment must be returned:
- Upon resignation or termination
- When no longer needed for role
- When receiving replacement equipment
- Upon request by IT or management

### Exit Process
**Employee Responsibilities:**
- Return all equipment on last day
- Equipment must be in working condition
- Remove personal files (if time permits and IT allows)
- Provide passwords for any personal accounts on device
- Return accessories (chargers, cables, adapters)

**Company Actions:**
- IT wipes device after employee separation
- Company data backed up (if not already in cloud storage)
- Licenses reclaimed for reassignment
- Equipment cleaned and refurbished for next user

### Equipment Checklist
Return all of the following:
☐ Laptop computer
☐ Power adapter and cables
☐ Mobile phone (if company-provided)
☐ Tablets or other devices
☐ Monitors
☐ Keyboard and mouse
☐ Headset
☐ Docking station or hub
☐ Laptop bag
☐ Security badges and access cards
☐ Keys
☐ Any other company property

### Failure to Return Equipment
- Final paycheck may be withheld pending equipment return (where legally permitted)
- Cost of unreturned equipment may be deducted from final pay
- Legal action may be taken for unreturned equipment
- Outstanding equipment may affect reference

### Data Ownership
- All data on company equipment is company property
- Company may access and retain data after separation
- Employees should not have expectation that personal files will be returned
- Personal data on company equipment should be backed up to personal storage before separation

## Bring Your Own Device (BYOD)

### BYOD Policy
Employees may use personal devices to access company email and limited resources.

### Requirements
**To participate:**
- Device must meet minimum OS requirements
- Enroll in Mobile Device Management (MDM)
- Accept MDM terms and conditions
- Enable device passcode/biometric lock
- Keep device updated with latest OS and patches
- Company data containerized (separated from personal data)

### What Can Be Accessed
- Email (Outlook app)
- Calendar and contacts
- Teams chat and calls
- View-only access to SharePoint documents
- Other approved mobile apps

### What Cannot Be Accessed
- Full VPN access to internal network
- Downloading confidential files for offline storage
- Administrative systems
- Development environments

### Company Rights
**With MDM enrollment, company can:**
- Enforce security policies on device
- Require device passcode
- Remotely wipe company data (NOT personal data)
- See device information (model, OS version, installed apps)
- Restrict certain device features for security

**Company CANNOT:**
- Access personal data, photos, or apps
- Wipe entire device (only company data container)
- Track location outside work context
- Monitor personal communications

### Employee Responsibilities
- Maintain device security
- Report lost/stolen device immediately
- Understand that company data may be remotely wiped
- Comply with all security policies
- Remove MDM profile upon separation (IT assists)

### Reimbursement
- Employees using personal devices for work receive monthly stipend
- Amount varies by role and usage
- Submitted via expense report

## Security Requirements

### Password and Authentication
- Use strong passwords/passphrases
- Enable multi-factor authentication (required)
- Don't share credentials
- Lock device when unattended (Windows+L or Cmd+Ctrl+Q)
- Set auto-lock timeout (5 minutes maximum)

### Encryption
- All company laptops encrypted with BitLocker (Windows) or FileVault (Mac)
- Mobile devices encrypted by default (modern devices)
- Do not disable encryption

### Antivirus and Security Software
- Antivirus installed and active on all devices
- Do not disable or uninstall security software
- Run periodic scans
- Report suspected malware immediately

### Physical Security
- Keep equipment secure (don't leave unattended in public)
- Lock laptop to desk with cable lock when in office
- Don't leave in visible in car
- Use privacy screen when working in public
- Report lost or stolen equipment immediately

### Remote Work Security
- Use VPN for all remote access
- Secure home network (WPA2/WPA3 encryption)
- Don't allow family members to use company equipment
- Secure workspace (don't leave equipment accessible to others)
