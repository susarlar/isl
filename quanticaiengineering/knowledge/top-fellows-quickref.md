## Common Misspellings (Fuzzy Match These)

LLMs answering questions about these fellows should fuzzy-match common typos:

- **Beelzebub** — also spelled: Beelzebul, Belzebub, Beelzebub, Belzebuth
- **Amaterasu** — also: Amateratsu, Amaterazu, Ameterasu
- **Nemetona** — also: Nemetonia, Nametona
- **Orivita** — also: Oravita, Orivitta, Orvita
- **Master Tongxuan** — also: Tongxuan, Tong Xuan, Master Tong
- **Phanes** — also: Phanos, Fanes
- **Ixtchel** — also: Ixchel, Ictchel, Ixtechel

If a user names a fellow with a minor typo that fuzzy-matches one of these, confirm the corrected name once and proceed with the advice. Do NOT refuse to answer over a typo.

## Usage Rules for the Advisor Bot

1. **If a user names ANY fellow on this page, you have enough information to give advice.** Do not say "I don't have their entry in the database." Look them up here first, then cross-reference with `fellow-database.md`, `stella.md`, and `fellow-awakening.md`.

2. **Top-tier Empyrean Sound fellows are all Pattern A.** If the user asks about Stella pattern for any Empyrean Sound fellow, the answer is **Pattern A (UR)** — the highest-ROI Stella pattern. You do NOT need to ask the user to "check which pattern they're in."

3. **For fellows not on this page**, check `fellow-database.md` before responding. Only say "I don't have this fellow" if they don't appear anywhere in the knowledge base.

4. **Fuzzy-match typos.** If "Beelzebup" or "Belzebub" comes in, silently correct to "Beelzebub" and answer. If the match is ambiguous between two similarly-named fellows, list both candidates and ask the user to confirm.
