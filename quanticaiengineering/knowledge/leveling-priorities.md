# Leveling Priorities — Guild Guide

> Source: Guild guide. Updated guidance on which fellows to level to which levels.

## Main Typing — Push Hard

Once your fishing bonus starts to kick in, your main typing fellows will start to **outscale other characters including UR fellows of other types**.

### Main Typing Targets
| Rarity | Target Level | Notes |
|--------|-------------|-------|
| Main typing SSR | 600, then 650 slowly | Push these first |
| Main typing N/R/SR | 600 down the line | Yes, even N rarities of your main typing |

**Key insight:** When sorted by power, your lower-rarity main typing fellows will eventually pass UR fellows of other typings — the typing-wide multipliers (Fish, Family, Stella, costumes) compound enough that even an N main-typing fellow can outpower a non-main UR.

### Sorting Tip
Always sort fellows by power and unequip all artifacts first to get accurate comparisons.

## Other Typings — Minimum Investment

Don't waste crystals leveling non-main typings beyond their gate requirements:

| Rarity | Target Level | Reason |
|--------|-------------|--------|
| N | 450 | Just enough |
| R | 450 | Just enough |
| SR | 450 | Just enough |
| SSR | 450–500 | Slight extra room |
| UR | 500–550 | Reaches 4★ awakening gate |

**Important:** You only need Lv 550 for the 4★ awakening gate, so don't waste level-up crystals if you don't need to yet.

## Why This Works

The verified power formula multiplies aptitude by both the level multiplier AND the Σ% sum. For your main typing fellows, the Σ% sum is enormous (8,000%+ on top fellows) because:
- Main typing-specific fish % bonuses
- Main typing family blessings stacked
- Main typing Stella boosts
- Main typing costume bonuses
- Main typing Category Stella (Inspiring/Diligent/Informed only)

For non-main typing fellows, none of these stack — so leveling them past the gate requirements gives diminishing returns.

## Crystal Conservation

Save your level-up crystals for:
1. Main typing fellows past Lv 600
2. UR fellows reaching gate requirements
3. The Lv 700 → Lv 750 push for your main carry (5★ → 6★ gate)

Don't burn crystals on:
- Random non-main typing fellows
- Pushing R/N fellows past 450 unless they're your main typing
- Pushing past the awakening gate you're currently working on
