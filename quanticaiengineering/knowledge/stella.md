# Stella System — Overview

> For complete level tables, costs, and member lists per group, see `stella-tables.md`.

## What Stella Is

Stella is a per-fellow upgrade system that provides:
1. **Personal Power boost** to the fellow
2. **Group-wide Power boost** to all fellows in the same Stella group
3. **Building Earnings** boosts (Pattern A only)
4. **Aptitude bonuses** to the group (Pattern A only)
5. **New aptitude slots** for skill pearls (Pattern A only)
6. **Blessing Bond** unlocks (Pattern A Lv4)
7. **Aptitude skill cap raises** (Pattern A only)

## All Fellows Have Group Stella

**Every fellow** improves their group's Stella with their own personal character shards — except for two groups that share a common currency:

| Currency | Group | Mechanic |
|----------|-------|----------|
| Purple Stars | Archdemons | Shared — any star upgrades any Archdemon |
| Insignias | Otherworld Valiants | Shared — any insignia upgrades any Valiant |
| Magi Shards | Ancient Magi | Shared but random from chests |
| Character Fragments | All other groups | Each fellow needs their OWN dupes |

This means Archdemons and Otherworld Valiants are far more efficient to push to high Stella levels than other groups (where each fellow is bottlenecked by their personal fragment income).

## Four Stella Patterns

| Pattern | Lv10 Group Boost | Used By |
|---------|-----------------|---------|
| **A: Premium UR** | +175% power, +140 apt, +400% earnings, slot unlocks | Empyrean Sound, Ancient Magi, Divine Gospel |
| **B: SSR High** | +50% (alternates group/fellow) | Archdemon, Otherworld, Elites, Diligent Dynamo, Moonlit Visitor, Uncrowned, Greyrat Family, Dungeon Challenger |
| **C: SSR Low** | +25% (alternates group/fellow) | Followers |
| **D: SR Standalone** | +30% to ALL of category | Rani, Elise, Liz, Angie (no group affiliation) |

These fellows have personal Stella that boosts an entire aptitude category, but they aren't part of any group.
- **Rani** (SR, Inspiring booster)
- **Elise** (SR, Inspiring booster)
- **Liz** (SR, Diligent booster)

**Brave and Unfettered have NO standalone Category Stella fellow.**

## Critical Strategic Insight

**Brave and Unfettered have NO SR Category Stella booster.** This is a permanent ~30% multiplier gap if you main these types.

Inspiring uniquely has TWO Category Stella boosters (Rani + Elise), and **they stack fully** — both Rani's +30% and Elise's +30% add to the Σ% pool independently for **+60% total** to all Inspiring fellows. This is verified in-game: there is no cap on having multiple Category Stella boosters active for the same type. Inspiring carries therefore have the strongest SR-sourced multiplier stack of any typing.

## Data Gaps — Higher Stella Levels

Our verified data covers:
- **Pattern A (UR groups):** Levels 1–20
- **Pattern B (SSR high):** Levels 1–20 (Archdemon extends to Lv 40)
- **Pattern C (Followers):** Levels 1–10
- **Pattern D (SR Category):** Levels 1–10

**Beyond these levels the exact values are not in this knowledge base.** If you have information about higher Stella levels (Pattern C above Lv 10, Pattern D above Lv 10, or Pattern A/B max caps), contact **Su** to get it added. Do not trust wiki numbers for high Stella — they're often wrong or outdated.

## Investment Priority

1. **Pattern A (UR Groups)** — Highest ROI per Stella level
2. **Pattern B with shared currency** (Archdemons, Otherworld Valiants) — Easy to push
3. **Pattern D (SR Category)** — Massive value if you main I/D/I (cheap SR fellows)
4. **Pattern B with personal fragments** (Elites, Dynamo, etc.) — Bottlenecked by dupes
5. **Pattern C (Followers)** — Lowest ROI, last priority
